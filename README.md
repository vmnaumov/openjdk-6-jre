# openjdk-6-armel

Kindle package with 
[openjdk-6 java virtual machine](https://github.com/vmnaumov/openjdk-6-armel/releases/download/v1.0.0/openjdk-6-jre.zip) 
for running java application on Kindle.

Package in not useful without java applications.

Downoad zip file and unpack to `extensions` folder.
